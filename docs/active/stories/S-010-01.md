**Scope:** A new pure module src/lib/boundary-contract.ts defining the BoundaryContract shape — declared path, key env name, assertShape(body), verify(key, body), and the page landmark (heading, status/body selectors, evidence selectors + patterns, primary-action name) — and exporting receiptBoundary, the receipt exemplar filling that shape, built from the pure receipt.ts helpers. Owns only the declaration artifact; no check script, core, flow spec, or route is edited here.

**Story acceptance:** Unit tests exercise receiptBoundary end to end: assertShape accepts a genuinely-signed receipt and throws on a wrong-boundary name, a missing/blank field, and a non-hex nonce/signature; verify returns true for a real signature and false for a corrupted one; the declared path, heading, selectors, and action name equal the receipt exemplar's current literals; and the module imports nothing from node:* (safe for both the Node scripts and the Playwright config to import).

**Honest boundary:** This story only DECLARES; it rewires no consumer. The Node harness (S-010-02) and the flow (S-010-03) keep reading their hardcoded /api/receipt, receipt-shaped assertions, and Demo Runway literal until pointed at this module — deferred there, not hidden here. All work is FREE and fixture-proven (pure crypto, no server, no tokens).
