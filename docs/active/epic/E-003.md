## Intent — the bigger-picture play

Give each demo a backstage door: a passcode-gated surface where nontechnical collaborators submit references, links, and feedback as structured entries, plus one documented seam through which the project's coding agents retrieve those entries verbatim. The template promises a public URL early; this play makes that URL two-way — stakeholder context stops living in chat screenshots and becomes retrievable project state, exactly when ideation is still cheap to steer.

## Value to the design

Realizes the charter's shared-passcode collaboration surface: structured stakeholder input becomes agent-retrievable state with zero account or workplace overhead on either side, which is the substance of P4 rather than a gesture at it. As a permanent, it is reusable leverage across every demo the template generates, not a one-demo fixture.

## Done looks like

A nontechnical stakeholder, given only the demo URL and the shared passcode, submits a structured reference or piece of feedback from a phone; a coding agent then retrieves that exact submission through a documented seam — and no account was created on either side.
