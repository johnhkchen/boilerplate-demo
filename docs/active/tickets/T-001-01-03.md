## Context

Removes deploy friction permanently: every push to main redeploys with no manual steps, so integration and deploy errors surface against the live site from day zero.

_Advances: P1_

## Acceptance Criteria

- [ ] Pushing a trivial visible change to main triggers the CI workflow, it finishes green, and the change is observable at the public URL with no manual step in between; deploy credentials exist only as CI secrets — never in the repository or the browser bundle.
